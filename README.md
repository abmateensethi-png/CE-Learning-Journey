CE-Learning-Journey: My Computer Engineering Experience 💻🌟 

This repository contains a detailed and visually organized collection of my reflections, articles and projects from my journey in Computer Engineering (CE). It spans from the 1st semester, where I started learning Python programming and Machine Learning models, to the 2nd semester, where I explored Object-Oriented Programming (OOP), Digital Logic Design (DLD) with labs and databases, along with their associated projects. This repository is designed to document my learning process, lab experiences, projects and key insights — perfect for students, educators or anyone curious about the step-by-step growth of a CE student.

🌱 My Learning Journey

1st Semester: Laying the Foundation
Python Programming: Learned the basics of Python syntax, control structures, functions and modules.
Machine Learning Models: Explored simple ML models like Linear Regression and Classification and trained models on datasets to understand predictions.
Practical Application: Worked on lab exercises and assignments to reinforce coding and analytical skills.
Key Reflection: The first semester was all about understanding programming logic, solving problems computationally and building confidence with small projects.

2nd Semester: Expanding Knowledge 
Object-Oriented Programming (OOP): Learned about classes, objects, encapsulation, inheritance and polymorphism. 
Digital Logic Design (DLD) and Labs: Designed and implemented logic circuits using tools and hands-on lab exercises. Databases: Learned to create, manage and query databases using SQL.
Projects: Applied learning in small projects combining programming, OOP, DLD and databases.
Key Reflection: This semester focused on structuring code, understanding real-world system design and integrating multiple skills in projects. 
Future Semesters: As I progress in CE, I plan to document new experiences, advanced projects and emerging technologies I explore. This repository will continue to evolve as a comprehensive learning diary.

📂 What’s Inside?

Markdown Articles: Each article details a concept, lab experience or project, along with personal reflections. 
Semester-wise Projects: Python scripts, OOP programs, DLD simulations and SQL queries. 
Step-by-step Learning: Each file documents my approach, mistakes and lessons learned.
Hands-on Exercises: Lab-based exercises and small projects demonstrating applied knowledge. 

🌟 Why This Repository?

This repository is a personal and educational record of my growth as a CE student. It shows how consistent practice, experimentation and reflection can help a student: Build strong programming and analytical skills Understand hardware and logic design concepts Integrate databases with applications Prepare for real-world CE projects It is ideal for students, instructors and peers to follow along, get inspiration and understand the journey of learning CE step by step.

⚡ Key Takeaways 

Learning CE is a step-by-step journey: start small, build gradually, reflect and improve. Lab exercises and practical projects are essential for understanding real-world applications. Documenting your journey helps consolidate knowledge and track progress.
